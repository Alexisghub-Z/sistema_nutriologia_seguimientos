# Sonidos de Notificación

Este directorio contiene archivos de audio para notificaciones del sistema.

## notification.mp3

Este archivo debe ser un sonido corto (0.5-1 segundo) para notificaciones de nuevos mensajes.

Puedes descargar sonidos libres de:
- https://notificationsounds.com/
- https://freesound.org/
- https://mixkit.co/free-sound-effects/notification/

O crear tu propio archivo de audio y colocarlo aquí con el nombre `notification.mp3`.
